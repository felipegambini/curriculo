Sobre

Atualmente sou Analista de Dados Senior em uma financeira.

Tenho 4 anos de experiência com Power BI, DAX avançado e Excel avançado;

Um pouco mais de 3 anos de experiência em python;

Em torno de 1 ano e meio de experiência em SQL.

Por ter pouco acesso aos bancos de dados na empresa, no inicio realizava o ETL no próprio power query, ao longo do tempo migrei tudo para python para melhorar a performance do ETL e dos painéis. Agora com mais abertura aos bancos, também utilizo muito para consulta e para construção de aplicações com dados quase em tempo real.

Comecei a estudar Git junto com Python e hoje utilizo muito no meu dia a dia para versionamento e colaboração de código, aprendi Git tanto em CLI quanto por interface do GitHub Desktop, sou fascinado pela documentações tanto do Git quanto Python e de outras ferramentas.

Comecei aprendendo Python para automação de processos, e após para tratamento de dados.

Hoje, grande parte do meu conhecimento é voltado para análise e tratamento de dados com pandas e duckdb. Mas já tive muita experiência com vários recursos que o python oferece (Flask, WebScraping, automação de tarefas, requests, TKinter e PySimpleGUI, REST APIs, ML).
